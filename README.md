# PersonalAIassisstant
A personal AI assistant (Samuel) with memory, chat interface, and Ollama integration.
# Samuel AI 🤖

## Features
- 🧠 Memory system (SQLite-based)
- 💬 Chat interface
- 🎙 Voice integration (WIP)
- 🎨 UI with panels (chat, memory, prompts)
- 🧠 Style learning system
- 🔍 Web search capability

## Tech Stack
- Python
- Tkinter (GUI)
- SQLite (memory system)
- Ollama (LLM)

## How to Run

```bash
pip install -r requirements.txt
python gui_app.py
